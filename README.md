# Thoughtful.ai technical screen

Technical screening to create a package sorting function. See [provided instructions](https://thoughtfulautomation.notion.site/Core-Engineering-Technical-Screen-b61b6f6980714c198dc49b91dd23d695) for more info

### Getting Started

1. Pull the repo `git checkout https://github.com/JordanMajd/thoughtful.ai.git`
2. Navigate into the repo `cd thoughtful.ai`
3. Install dependencies `yarn install`
4. Run unit tests `yarn run test`
